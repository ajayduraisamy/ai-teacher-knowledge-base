# AI Teacher Knowledge Base

A structured, production-ready educational repository containing **5000+ concepts** spanning Mathematics, Machine Learning, Deep Learning, Computer Vision, NLP, LLMs, AI Agents, MLOps, Cloud Computing, and System Design.

Each concept is a self-contained markdown file with definitions, intuition, mathematical formulations, AI/ML relevance, worked examples, practice problems, and interview questions — designed to serve as a complete reference for learners, educators, and practitioners.

---

## Vision

To build the most comprehensive, well-structured, and pedagogically sound open-source knowledge base for AI and its mathematical foundations — accessible to beginners yet rigorous enough for advanced learners.

## Goals

- **5000+ concepts** across 10+ domains, each with consistent structure and depth.
- **Mathematical rigor** with intuitive explanations — every formula explained, every step shown.
- **AI/ML relevance** woven into every concept — connecting theory to real-world practice.
- **Interview-ready content** — curated questions from beginner to advanced.
- **Modular and extensible** — designed so new concepts, modules, and domains can be added without restructuring.
- **Production quality** — clean markdown, proper LaTeX, consistent formatting, and cross-referencing.

## Contribution Guidelines

We welcome contributions! To add or improve a concept:

1. **Follow the template** — every concept must include all sections listed above.
2. **Use LaTeX** — enclose inline math in `$...$` and display math in `$$...$$`.
3. **Be self-contained** — each concept should be understandable without external references.
4. **Include AI/ML relevance** — connect every concept to practical AI/ML use cases.
5. **Show all steps** — worked examples must show intermediate calculations.
6. **Cross-reference** — link to prerequisite and next concepts using their IDs.
7. **Name files by ID** — use zero-padded three-digit IDs (e.g., `001_scalar.md`).
8. **Submit via PR** — one concept per PR is preferred for easier review.
