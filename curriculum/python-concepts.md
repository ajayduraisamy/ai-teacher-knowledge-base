# Python Curriculum — 100 Concepts

## Overview

Comprehensive Python programming curriculum covering fundamentals through advanced topics, data science libraries, and machine learning frameworks. Each concept is a self-contained markdown file with definitions, examples, and practical applications.

**Total Concepts:** 100
**Difficulty Range:** Beginner → Intermediate → Advanced

## File Naming Convention

Files are named with zero-padded three-digit ID and lowercase underscore slug:

```
{id:03d}_{slug}.py.md
```

Examples:
```
001_variables_and_data_types.py.md
026_classes_and_objects.py.md
093_scikit_learn_basics.py.md
```

## Directory Structure

```
python/
├── 001_variables_and_data_types.py.md
├── 002_numbers_and_arithmetic.py.md
├── ...
└── 100_python_project_structure.py.md
```
